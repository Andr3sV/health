## Getting Started

### Development

```bash
npm install
npm run dev
```

Open [http://localhost:3083](http://localhost:3083) in your browser.

### Production Build

```bash
npm run build
npm start
```

## Deployment

### Deploy to Vercel

1. Push your code to GitHub
2. Go to [Vercel](https://vercel.com)
3. Click "New Project"
4. Import your repository
5. Vercel will automatically detect Next.js and configure the project
6. Click "Deploy"

The deployment will be ready in a few minutes!

### Environment Variables

No environment variables are required for this project.
